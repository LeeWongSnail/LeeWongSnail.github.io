    • 在我们日常的开发工作中,我们会经常使用一些快捷键,其中的一些快捷键可以很大程度上提高我们开发效率,所以在此对常用的快捷键进行整理！

    删除一行代码

    ctrl+k 删除光标之后的内容
    如果要删除整行代码 那么需要保证光标在行首

    将光标跳转至行首

    ctrl+a
    cmd+<- cmd+-> 光标跳转至行首或者行尾

    查看工程文件

    command+1

    全局查找

    command+3

    快速跳转到类的特定行

    command + L

    快速查找打开类

    command+ shift+ O

    快速定位当前类在项目文件中的位置

    command + shift + J

    类文件”.h”与“.m”之间切换

    control+command+↑/control+command+↓

    清除工程

    Command + Shift + K

    当前类 方法/变量查找

    Control+6（键入方法/变量名+Enter跳转）

    撤销/反撤销：

    command+z /command +shift +z

    光标操作

    向右一个字符（forward）

    control+F

    向左一个字符（backward）

    control+B

    前一行（previous）

    control+P

    后一行（next）

    control+N

    去行首

    control+A

    到行尾（end）

    control+E

    删除光标右侧字符（delete）

    control+D

    整体位移代码:

    向左位移

    选中的代码 + command+[

    向右位移

    选中的代码 + command+]

    向上位移

    选中的代码 + command+option+[

    向下位移

    选中的代码 + command+option+]

    xcode 整体

    回到上一个/下一个打开的页面

    control + command + ← / control + command + →

    新建文件

    command + N

    创建新工程:

    command + shift + N
