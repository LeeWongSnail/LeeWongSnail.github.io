iOS LLDB 打印技巧

发表于 2015-12-19 更新于 2020-07-19 分类于 iOS进阶 阅读次数：

今天在微博上看到小笨狼分享的一片文章，其中比较详细的介绍了LLDB的使用，顿时感觉原来LLDB控制体可以做那么多事情，下面是我挑选的几个，开发中比较常用的命令

1、基本语法

首先还是先看一下最近本的语法，这会让我们会更容易的去理解和记住下面的命令

breakpoint set - n main

br s -n main

expression –
